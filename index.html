The Gist

This library is nothing more than a collection of really common UI snippets and a utility class or two.

Features

We created the Axe\Page class which allows different code to add add links, metas, scripts, and such to an HTML page's head or body. Using a design pattern like Visitor or Listener allows objects completely unaware of rendering to influence generated markup.

We've also included several helper tags for common stuff.

  • Gravatar image
  • Columns (and golden ratio columns)
  • Bootstrap-compatible breadcrumbs, form groups, and fieldsets
  • Nearly Bootstrap-compatible alerts
  • Email and Phone Number links
  • Convert text with line endings to multiple <p>s
  • <select> and <option>s generated from a Map of values
  • …and more!
